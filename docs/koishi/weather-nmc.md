# weather-nmc 说明文档

`weather-nmc` 的说明文档，解释了用法和一些其他问题


## 这插件是干啥的？

用于获取天气。一开始准备用nmc的，但是后来换成了百度。但是因为已经发包了所以懒得改名字了。

### 用法

```sh
.天气 <城市名>
```
### 示例

查询杭州天气

```sh
.天气 杭州
```
